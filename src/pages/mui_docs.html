Motion UI Docs

Welcome!

These are the docs for the alpha version of the Foundation for Apps Motion UI. The goal here is to allow you to rapidly prototype animated elements that integrate seamlessly into your site!

{{#markdown}} #### The Basics: Animating an Element Quickly addding an animation to an element is easy! We use the class `animate` to animate an element, and we use the attribute `data-animation` to describe how we want it to move. {{/markdown}}
{{#markdown}} ###### Animations You can choose from a number of prefabricated animations: `slide-in-from-north`, `slide-in-from-east`, `slide-in-from-south`, `slide-in-from-west`, `slide-out-to-north`, `slide-out-to-east`, `slide-out-to-south`, `slide-out-to-west`, `fade-in`, `fade-out`, `shake`, `spin-cw`, `spin-ccw`, `wiggle` ###### Markup ```html

{{#markdown}} #### Modifiers Don't want an out-of-the-box animation? No problemo! You have several modifiers at your disposal to get exactly the animation you want! {{/markdown}}
{{#markdown}} ##### Speed You have three speeds to choose from. By default, animations will execute in **700ms**. You can pass `slow` or `fast` modifiers to change the speed to **1.5s** or **300ms** respectively. ###### Markup ```html
