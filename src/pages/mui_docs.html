Motion UI Docs

Welcome!

These are the docs for the alpha version of the Foundation for Apps Motion UI. The goal here is to allow you to rapidly prototype animated elements that integrate seamlessly into your site!

Would you like to know more?

{{#markdown}} #### The Basics: Adding an Animation to an Element Quickly addding an animation to an element is easy! We use the class `animate` to animate an element, and we use the attribute `data-animation` to describe how we want it to move. {{/markdown}}
{{#markdown}} ###### Markup ```html
``` You can choose from a number of prefabricated animations: - `slide-in-from-north` - `slide-in-from-east` - `slide-in-from-south` - `slide-in-from-west` - `slide-out-to-north` *todo* - `slide-out-to-east` - `slide-out-to-south` - `slide-out-to-west` - `fade-in` - `fade-out` - `shake` - `spin-cw` - `spin-ccw` - `wiggle` {{/markdown}}
